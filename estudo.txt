Primeiros comandos GIT:
1. Para configurar o GIT em uma máquina nova, é preciso utilizar o comando "git config --global user.name "nome de usuário".
2. Utilizar o comando git config --global user.email "email" para configurar o email referente ao usuário.
3. Após configurar o GIT na máquina, deve ser criado um novo repositório utilizando o comando mkdir nome do repositório
4. Após criado o repositório deve ser selecionado no GIT o respectivo repositório utilizando o comando "cd nome do repositório"
5. Após selecionado o repositório no GIT, é preciso iniciar o repositório, utilizando o comando "git init"
6. Assim o GIT estará pronto para começar a fazer edições de texto utilizando o vim nome do projeto.

Após ter terminado o projeto-

Comandos básicos ao fazer e terminar o projeto:

1- git status - verifica em que situação o projeto se encontra, untracked, unmodified, modified ou staged.
2- git add nome do projeto - adiciona o arquivo ao repositório GIT, tornando o arquivo modified.
3. git commit - commita o arquivo tornando ele como staged.
4. git log - mostra os commits feitos e mensagens do que foi alterado naquele commit.
5. git log --graph - mostra em gráficos os commits feitos.
6. git diff - mostra o que foi alterado em um projeto apenas antes dele ser commitado.
